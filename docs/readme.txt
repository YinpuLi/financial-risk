数据描述数据分为两部分：
训练数据及测试数据，均来源于行内额度授信审批过程中的批复数据，审批意见文本内容中企业名称、业务编号等内容已经过脱敏处理。
文件清单和使用说明
训练数据包含1个文件：
train.csv。该文件包含三个字段：

ID：唯一标识；
content：批复意见文本内容；
label：是否有财务风险。

<p>训练集为带有财务风险标签值（label）的数据集，其中label=0表示无财务风险描述，为1则表示有财务风险描述。训练集的标签不完全准确，且有些是基于业务提出的一些规则进行了处理。</p>
<p><strong>测试数据包含1个文件：</strong></p>
<ol>
<li>测试A榜：testA.csv；</li>
<li>测试B榜：testB.csv，在复赛阶段开放。</li>
</ol>
<p>测试数据文件字段为ID（唯一标识）、content（批复意见文本内容），无label字段。A榜、B榜两份数据，分别用于初赛和复赛阶段。</p>
